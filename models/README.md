# Models

Trained model artifacts are not included in this repository due to file size.

To reproduce all models, run the full pipeline via:
- master_training_script.py — end-to-end pipeline in one execution
- notebooks/01_synthetic_baseline.ipynb — Session 1 Kaggle notebook
- notebooks/02_nasa_mdp_real_data.ipynb — Session 2 Kaggle notebook

All notebooks were developed and executed on Kaggle free CPU. No GPU required.
